01 작성 방향

챕터 구조

  • ✓ 챕터당 1,500–3,000 단어 권장
  • ✓ 매 챕터 끝 클리프행어 또는 감정적 비트
  • ✓ 첫 3챕터 내 두 주인공 첫 만남 배치
  • ✗ 초반 인물 설명 나열 지양 — 행동으로 보여줄 것

로맨스 공식

  • ✓ HEA(Happily Ever After) 또는 HFN 필수
  • ✓ 명확한 감정 호선(emotional arc)
  • ✓ 주인공 내면 독백 풍부하게
  • ✗ 열린 결말·비극 엔딩은 장르 규칙 위반

시점 & 문체

  • ✓ 1인칭 또는 3인칭 제한시점 주류
  • ✓ 듀얼 POV 구성 특히 인기
  • ✓ 현재형(present tense) 트렌드 증가
  • ✗ 3인칭 전지적 시점은 구식으로 인식

캐릭터 설정

  • ✓ 여주 독립적·주도적인 캐릭터 선호
  • ✓ 남주는 결함(flaw)이 있어야 입체적
  • ✗ 지나치게 수동적인 여주는 독자 이탈
  • ✗ 독성 남주를 로맨틱하게 묘사 시 비판
1인칭 선호 듀얼 POV 인기 현재형 트렌드 HEA 필수 slow burn 수요 높음
02 문화 주의사항
⚠ 한국 웹소설에서 당연한 설정이 미국 독자에게 불쾌감 또는 혼란을 줄 수 있습니다. 아래 항목을 반드시 검토하세요.

피해야 할 설정

  • ✗ 파워 불균형 강조 (상사-부하, 재벌-평민)
  • ✗ 쌍꺼풀·흰 피부를 미의 기준으로 서술
  • ✗ CEO-인턴 로맨스 — #MeToo 맥락 민감
  • ✗ "싫다는 게 좋다는 것" 식 구애 클리셰
  • ✗ 성별 고정관념 고착화

선호되는 요소

  • ✓ 명확한 동의(consent) 표현
  • ✓ 여주의 직업·목표 독립성
  • ✓ 건강한 소통으로 갈등 해결
  • ✓ 다양성(인종·체형) 포용적 묘사
  • ✓ 트라우마를 제대로 다루는 서사

문화 번역 필요 항목

  • ! 재벌 설정 — tech billionaire로 치환하거나 배경 설명 보완
  • ! 눈치 문화 — 비언어적 소통 장면에 내면 독백으로 보완
  • ! 음식·술자리 문화 — 독자에게 낯선 요소는 짧은 묘사 추가
