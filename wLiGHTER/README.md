# w.LiGHTER Workspace v14

## 실행
1. VS Code에서 이 폴더를 엽니다.
2. `index.html` 우클릭 → `Open with Live Server`로 실행합니다.
3. 왼쪽 하단 사용자 설정 아이콘에서 OpenAI API Key를 저장합니다.

## 이번 버전 점검 범위
- 작품 목록 / 작품 등록 / 작품 정보 수정 / 작품 삭제
- 회차 목록 / 회차 등록 / 회차 삭제
- 번역·검수 작업 화면의 원문/번역본/번역 리포트 탭 전환
- 번역·검수 OpenAI API 호출 및 결과 표시
- 검수 챗봇 OpenAI API 호출 및 수정 제안 표시/반영
- 캐릭터 이미지 생성 결과 누적, 썸네일 전환, 선택 삭제
- 관계도 이미지 생성 결과 누적, 썸네일 전환, 선택 삭제
- 현지화 가이드 리포트 표시
- RAG JSON 및 문화권 검수 프롬프트 파일 포함

## 주의
이 버전은 내일 시연용으로 백엔드 없이 브라우저에서 직접 API를 호출합니다. GitHub 업로드 전 API 키를 반드시 삭제하세요.
